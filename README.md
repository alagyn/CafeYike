The **STACK**

```
-------------
| Bootstrap | Styling

      |
-------------

-------------
      |
-------------
|   Axios   | JS REST actions
-------------
      |
-------------
|  FastAPI  | Routing and API
-------------
      |
-------------
|   sqlite  | Yikebot Database
-------------
```

This is a Python household, the JavaScript will be kept to a minimum and all backend routing/API is handled by FastAPI.
